# Frontend

HTML + Vanilla JavaScript dashboard for the E-Wallet application.

## Pages

- `index.html` - Login page
- `wallet.html` - Main dashboard with balance and quick actions
- `transaction.html` - Transaction history and send money

## Features

- User authentication
- Wallet balance display
- Top-up and withdrawal
- Money transfer between users
- Transaction history
- Real-time notifications
- Responsive design

## API Integration

- Connects to API Gateway on port 3000
- Uses fetch API for all requests
- Real-time updates every 5 seconds

## Usage

Open `index.html` in a web browser and login with a user ID.
